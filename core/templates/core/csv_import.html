üìã CSV Format Instructions

Your CSV file should contain student names, one per row:

  • First row can be a header (like "Full Name" or "Name") - it will be automatically skipped
  • Each subsequent row should contain one student's full name
  • Names should be in format: First Middle Last or First Last
  • The system will automatically generate:
    • Student ID: ITC3001, ITC3002, etc.
    • Username: Based on name (e.g., "john.doe" ‚Üí "johnd", "johnd1" if exists)
    • Password: Derived from name with numbers (e.g., "johdo123")
